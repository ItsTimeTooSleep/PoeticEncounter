## 📖 使用指南

### 收听步骤
1. 打开 **Poem** 主文件夹 📁
2. 选择您想聆听的组员文件夹（张翔、张家齐、李建葳、伊仁魏卿）
3. 进入文件夹后，您将看到三首诗歌朗诵作品：
   - 🎨 两首**自创诗歌**（标注【原创】）
   - 🌟 一首**推荐诗歌**（标注【推荐】及原作者）
4. 点击音频文件即可在线播放或下载收听

### 文件说明
- 📝 文件命名包含作品类型及诗歌名称
- 🎧 音频格式：MP3 / m4a（高质量音质）
- ⏱️ 总时长：约10分钟

## 🎨 创作特色
